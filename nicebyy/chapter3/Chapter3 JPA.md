# Chapter3 JPA

# ORM 개요

---

> OOP 시대에 객체를 RDBMS에 관리하다 보니 SQL 중심적인 개발을 하게 됨으로써 나타나는 문제점들을 해결하기 위해 도입.

객체는 ↔ RDBMS 사이를 매끄럽게 연결하여 **SQL 의존성을 덜어내고 OOP 중점적으로 개발**을 하는 것이 목표.
>
